Detailed Description

Contains instances of every entity, including player and background. This class contains instances of every entity. It has a function to update positions of every entity, including player, taking collision into account. It also has a draw() function to draw all entities at once. Furthermore, it contains 4 vectors to determine direction, to be built into a seperate class.
