Detailed Description

A bullet object is an entity that`s always moving in one direction.

the buller class contains the item graphic and overrides the necessary functions from Entity. the direction, speed and rotation will be calculated in the constructur. when the bullet is inactive it will stop drawing and calculating the next position. if the bullet exist longer than 15 seconds it will go automaticly inActive
